# Goal

The users overall goal and objective for what they wanted grilling on. This is not "The user wanted to be challenged on feature x implementation" but instead "The user wants to implement feature x". 1 or 2 sentences.

## Shared Understanding Reached

* Dot points of what the shared understanding is - usually 5-10 in total. Don't include all of the details and Q+A here, still a summary.

## Immediate next steps

* Immediate next steps to take, if any.
* May include skills to invoke.

## Questions and Answers

### Question Name

*Options:*
* SELECTED: Option that was selected - always first.
* Alternative options that weren't selected (if any)

*Reasoning:*
Why that option was selected over alternatives

## Links/references

* Any useful links/references in dot-point format, can be local files, git branches, urls etc. Has a quick description at the end of it. 
* [Docs](www.mydocs.com): Includes...
